#5 — Description — Address Detection
Improved address detection
UPS Stores and commercial properties are prohibited as billing addresses for KYC. Users add these to ensure card delivery. Detecting these during KYC — not after — prevents account restrictions and reduces compliance paperwork. 1% of users have an impermissible address. 10k signups = 100 restrictions eliminated.

#8 — Description — Finding Stranded Users
Finding our stranded users
Users were failing document submission for various reasons — blurry photos, expired IDs, bad lighting. Previously, they were put in a failed state and forced to contact CS. New rules now detect and action these bad submissions proactively, allowing users to immediately resubmit.

#15 — Description — Introducing KYC Pending State
Introducing the KYC Pending state
Previously prohibited users can now submit their selfie and ID. Their account stays in a pending state during review. Users are informed and notified automatically — no prompts to contact CS. If determined to be a false positive, the user is promoted to KYC Success.
